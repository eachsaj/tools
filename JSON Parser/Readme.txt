Parses Json using Java

For program explanation visit:
https://cooltrickshome.blogspot.in/2016/11/creating-json-parser-using-java.html

Pom Dependency:
 <dependency>  
   <groupId>org.json</groupId>  
   <artifactId>json</artifactId>  
   <version>20160810</version>  
 </dependency>
