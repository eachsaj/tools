This program will help you convert a base64 string to an image and then save the image to the current directory using Java.

For more information visit:
https://cooltrickshome.blogspot.in/2016/11/convert-base64-encoded-string-to-image.html

Pom Dependency:
Please add the below pom dependency 
<dependency>
    <groupId>commons-codec</groupId>
    <artifactId>commons-codec</artifactId>
    <version>1.10</version>
</dependency>
