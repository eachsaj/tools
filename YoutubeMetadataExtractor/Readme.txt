This blog post will extract metadata information from youtube using Java. It makes use of JSoup library.

For more information visit:
https://cooltrickshome.blogspot.in/2016/12/create-youtube-metadata-crawler-using.html

Pom Dependency:
Please add the below pom dependency:
  <dependency>  
       <groupId>org.jsoup</groupId>  
       <artifactId>jsoup</artifactId>  
       <version>1.8.3</version>  
  </dependency>
